# 💰 Task 02 - Price Range Analysis

## 🎯 Objective
To analyze restaurant price ranges and study their relationship with aggregate ratings and rating colors.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📌 Steps Performed
1. Loaded the dataset
2. Explored price range distribution
3. Identified the most common price range
4. Calculated average ratings for each price range
5. Visualized average ratings across price ranges
6. Identified rating colors associated with price ranges
7. Determined the color representing the highest average rating

## 💡 Key Insights
- Certain price ranges are more common among restaurants  
- Average ratings vary across different price categories  
- Higher price ranges tend to have better ratings in some cases  
- Rating colors help visually represent restaurant quality
